Embed on your website

Paste this snippet before </body> on any page.

<!-- SupportFlow Chat Widget -->
<script>
  window.SupportFlowConfig = {
    apiKey: "YOUR_API_KEY",       // from Dashboard → API Keys
    baseUrl: "http://localhost:3000", // your SupportFlow domain
    primaryColor: "#6366f1",       // brand color
    position: "bottom-right",      // or "bottom-left"
    theme: "light",               // "light" or "dark"
    welcomeMessage: "Hi! How can we help you today?",
  };
</script>
<script src="http://localhost:3000/widget.js" async></script>
How to integrate

3 steps, under 2 minutes.

  1. Get your API Key

    Log into your SupportFlow dashboard → go to API Keys → copy your key.

  2. Paste the snippet

    Add the code above to your website's HTML just before </body>. Replace YOUR_API_KEY with your real key and baseUrl with your production domain.

  3. You're live!

    The widget loads automatically. Incoming chats appear in your Live Chat inbox. Reply from any device — desktop or mobile.
